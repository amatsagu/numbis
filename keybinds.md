## Added shortcuts
* Shift+Q: Closes the active window.
* Shift+B: Launches Dolphin.
* Shift+D: Launches KRunner (ideally if it can be centered on screen).
* Shift+Return: Launches your configured Terminal.
* Shift+[Arrows/HJKL]: Moves focus between windows.
* Shift+Alt+[Arrows/HJKL]: Swaps windows (currently uses index-based swapping for the master-stack layout).
* Shift+F: Toggles Fullscreen.
* Shift+Alt+F: Toggles Floating (centers the window upon floating).
